Summary

Multivrs is the name for a one-man game project that was in development for a year. It is made in Unreal Engine and mostly all 3D models are done in Houdini. The game has strategic and shooter gameplay elements and is about tracking down an adversary across the multiverse. There are items and gear to pick up in a lifeless and hostile world. The environment is using 3D procedural art, ensuring each game session to be unique.
